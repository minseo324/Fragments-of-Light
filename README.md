# 📝 마크다운 에디터

**무료 온라인 마크다운 에디터**에 오신 것을 환영합니다!

## ✨ 새로운 기능

### 📁 문서함
- 폴더로 여러 문서를 깔끔하게 관리
- 이름이나 내용으로 빠른 검색
- 툴바의 **폴더 아이콘** 클릭!

### 🖼️ 이미지 업로드
- 드래그 앤 드롭 또는 `Ctrl+V`로 붙여넣기
- 클라우드에 자동 업로드

### 💾 자동저장
- `Ctrl+S`로 저장, 30초마다 자동 저장
- Undo/Redo 완벽 지원 (`Ctrl+Z` / `Ctrl+Y`)

### 🎨 코드 하이라이팅
```javascript
function greet(name) {
  console.log(`안녕하세요, ${name}님!`);
}
```

### 🔄 스크롤 동기화
에디터와 미리보기가 함께 스크롤됩니다.

### 🌐 문서 공유
클릭 한 번으로 공유 링크 생성.
